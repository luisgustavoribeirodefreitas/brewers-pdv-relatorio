# Brewers PDV

Sistema de ponto de venda desenvolvido para uma cafeteria, com fluxo de atendimento para cliente e painel administrativo para equipe interna.

O projeto foi criado a partir de um protótipo no Figma e implementado como uma aplicação web utilizando **HTML, CSS, JavaScript e Supabase**.

---

## Sobre o projeto

O **Brewers PDV** simula o funcionamento de um sistema de cafeteria, permitindo que clientes realizem pedidos pelo cardápio digital e que atendentes acompanhem pedidos, mesas, clientes, pagamentos e produtos pelo painel administrativo.

O sistema possui duas áreas principais:

* **Área do cliente:** cardápio digital, detalhes dos produtos, carrinho e confirmação de pedido.
* **Área do atendente/administrador:** gerenciamento de pedidos, mesas, clientes, produtos, caixa e pagamentos.

---

## Funcionalidades

### Cliente

* Visualização do cardápio por categorias
* Detalhamento dos produtos
* Adição de itens ao pedido
* Carrinho de compras
* Confirmação de pedido
* Chamada de garçom

### Atendente/Administrador

* Login de funcionário
* Visão geral dos pedidos
* Controle de status dos pedidos
* Gerenciamento de mesas
* Cadastro e listagem de clientes
* Administração de produtos
* Abertura e fechamento de caixa
* Controle de pagamento dos pedidos

## Banco de dados

O projeto utiliza o **Supabase** como backend, com tabelas para armazenar informações como:

* produtos
* mesas
* clientes
* pedidos
* itens dos pedidos
* estado do caixa
* usuários da equipe
