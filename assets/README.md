# Assets a exportar do Figma

Coloque os arquivos exportados nesta pasta usando os nomes abaixo. O `script.js` já aponta para estes caminhos.

## Logos e ícones

- `logo-brewers.svg`: logo circular da tela `1 — Boas-vindas`, nó `Logo`/`Group`, exportar em SVG.
- `icon-trash.svg`: ícone de lixeira dos itens do carrinho, nós `Group`, exportar em SVG.
- `icon-bell.svg`: ícone da tela `6 — Chamar Garçom`, nó `Ícone Sino`, exportar em SVG.
- `icon-success.svg`: check da tela `5 — Pedido Confirmado`, nó `Ícone Sucesso`, exportar em SVG.
